# My AWS S3 Portfolio Website

## 🌐 Live Demo
🔗 [Click Here to View Website](http://jyoti-portfolio-website.s3-website.ap-south-1.amazonaws.com)

## 📌 Project Overview
This is my personal portfolio website hosted on **AWS S3** as a static website. It showcases my skills, projects, and experience as a **Software and Cloud Engineer**.

## 🛠️ Technologies Used
- **AWS S3** - Static website hosting
- **HTML5, CSS3, JavaScript** - Frontend development
- **Git & GitHub** - Version control

## 🚀 Features
- Responsive portfolio design
- Skills and services section
- Project showcase (Face Recognition Attendance System)
- Smooth scrolling and animations

## 📂 Project Structure

aws-s3-portfolio/
├── index.html # Main HTML file
├── style.css # Styling
└── script.js # JavaScript interactions


## 🏗️ How I Deployed This
1. Created HTML/CSS/JS portfolio website
2. Created S3 bucket `jyoti-portfolio-website`
3. Enabled static website hosting
4. Uploaded files to S3 bucket
5. Added bucket policy for public access
6. Website live! 🎉
